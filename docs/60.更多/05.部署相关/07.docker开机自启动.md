### 查看已启动的服务

```
systemctl list-units --type=service
```

### 查看是否设置开机启动

```
systemctl list-unit-files | grep docker
```

### 设置开机启动

```
systemctl enable docker.service
```

### 关闭开机启动

```
systemctl disable docker.service
```

### 现在启动docker

```bash
systemctl start docker
```
