Elasticsearch-aggregations聚合分析

# 前言

前面学习了如何通过Query DSL检索数据，接下学习如何将数据进行聚合分析。

# 简介

聚合提供了从数据中分组和提取数据的能力。最简单的绝活方法大致等于SQL GROUP BY和SQL聚合函数。

在Elasticsearch中，您有执行搜索会犯hits（命中结果），并且同时返回聚合结果，把一个响应中的所有hits（命中结果）分隔开的能力。这是非常强大且有效的，您可以执行查询和多个聚合，并且在一次使用中得到各自的（任何一个的）返回结果，使用一次简洁和简化的API来避免网络往返。

# 聚合语法

"aggregations" : {
    "<aggregation_name>" : {
        "<aggregation_type>" : {
            <aggregation_body>
        }
        [,"meta" : {  [<meta_data_body>] } ]?
        [,"aggregations" : { [<sub_aggregation>]+ } ]?
    }
    [,"<aggregation_name_2>" : { ... } ]*
}
